SOFTWARE GUIDE

Welcome to the OCR software guide! This guide will walk you through the process of using our web portal to extract text from images and PDFs using Optical Character Recognition technology. Our software provides advanced features like auto-correction and the ability to select specific portions of an image for text extraction.

INTRODUCTION

Optical Character Recognition (OCR) is a technology that enables the extraction of text from images and scanned documents. Our OCR software leverages cutting-edge algorithms to provide accurate and efficient text recognition. This software guide will help you make the most of our OCR web portal and its powerful features.

Supported File Formats

Our OCR web portal supports a wide range of file formats for text extraction. You can upload the following file types:
  • Images: JPEG, PNG etc.
  • PDF Documents: Standard PDF format

System Requirements

To use our OCR software, ensure that your system meets the following requirements:
  • Operating System: Windows 10, macOS, Linux, or compatible
  • Web Browser: Chrome, Firefox, or compatible

Uploading Files

To extract text from images or PDFs, follow these steps:
  1. On the dashboard, click the "Upload Files" button.
  2. Select the files you want to process from your local system.
  3. Wait for the upload to complete. The duration may vary depending on file size and internet speed.

Extracting Text

Once the files are uploaded, you can extract text from them using the following steps:
  1. Click on the "Extract Text" button next to the file.
  2. Our OCR software will analyze the file and extract the text content.
  3. Once the extraction is complete, the extracted text will be displayed on the screen.

Auto-Correction

Our OCR software incorporates an auto-correction feature to enhance the accuracy of extracted text.
  1. Our software will perform automatic corrections based on common patterns and language rules.
  2. Review the corrected text and make any necessary manual adjustments if required.

Selective Text Extraction

In addition to extracting text from the entire document, our OCR web portal allows you to select specific portions of an image for text extraction. Follow these steps:
  1. Open the image containing the desired portion of text.
  2. On the OCR web portal, click on the "TAKE SNIP" option.
  3. Use your mouse to draw a rectangle around the desired portion of the image.
  4. Click on the "Extract FROM SNIP" button.
  5. The OCR software will analyze the selected portion and extract the text.
